# cloneCoding-Zero

zero-cho강의 clone 코딩

인스타를 만들어서 crud를 배우는 강의로 

강의에는 
- 로그인 회원가입
- 글 올리기
- 이미지 올리기
- 글 수정하기 
- 해시태그 작성
- 팔로우 팔로잉
- 좋아요 
- 댓글 작성
등 sns을 만드는 수업으로 진행이 되었다.


프론트는
- antd를 통해서 디자인이 이루어지고 부가적으로 styled-componet를 사용해서 진행이 되었다.
- next를 이용하였다.
- 비동기는 saga

백엔드는
- express를 통해서 서버 제작
- db는 mySQL를 통해서 제작
